# 🚀 TwitchFreeTurbo - Installation Guide

Follow these steps to install the extension in under 60 seconds:

1.  **Download** the project by clicking the green **Code** button and selecting **Download ZIP**.
2.  **Extract** the ZIP archive to a folder on your computer.
3.  Open **Google Chrome** and navigate to `chrome://extensions/`.
4.  Enable **Developer Mode** using the toggle in the top right corner.
5.  Click the **Load unpacked** button.
6.  Select the extracted folder (the one containing the `manifest.json` file).
7.  **Pin** the TwitchFreeTurbo icon to your browser toolbar for easy access!

---
*Note: If the extension stops blocking ads, ensure you have the latest version from this repository and refresh the extension in the Chrome settings.*
