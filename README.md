## English

**TwitchFreeTurbo** is a high-performance Chrome extension designed for a seamless, interruption-free Twitch experience. It combines advanced bypass techniques with essential quality-of-life tools for power viewers.

### ✨ Turbo Features

#### 🛡️ Ad Protection (Triple Layer)
- **Thunderdome Bypass**: Intercepts stream requests to prevent pre-roll and mid-roll ads from loading.
- **Automatic 360p Mode**: During ad sequences, the extension switches to an ad-free stream (temporary quality reduction) so you never miss the action.
- **Picture-in-Picture Mode (SDA)**: If Twitch forces a full-screen ad, the stream is automatically brought to the front while the ad is minimized and muted.

#### ⚡ Power Tools
- **Instant Rewind**: A `⏪` button added directly to the Twitch player to jump back 10 seconds in one click.
- **Auto-Claim Points**: Automatically collects channel points (bonus chests) as soon as they appear.
- **Real-Time Stats**: A built-in counter to track how many ads have been blocked.

#### 🎨 Premium UI
- Modern, minimalist design perfectly integrated with Twitch's Dark Mode.
- Subtle notifications to confirm extension actions.

### 🛠️ Quick Installation

This extension is in developer mode. Here’s how to install it in under 60 seconds:

1.  **Download** the project (Green `Code` button > `Download ZIP`).
2.  **Extract** the ZIP archive to a folder on your computer.
3.  Open your browser and go to `chrome://extensions/`.
4.  Enable **Developer Mode** (toggle in the top right).
5.  Click on **Load unpacked**.
6.  Select the extension folder (the one containing `manifest.json`).
7.  **Pin** the TwitchFreeTurbo icon to your toolbar!

### 💡 Good to Know
*   **Video Quality**: When bypassing an ad, it is normal for the quality to temporarily drop (360p/480p). it will automatically return to normal as soon as the ad segment ends for other users.
*   **Updates**: If the extension stops blocking ads, check for updates on this repository and refresh the extension in Chrome.

### 🛠️ Installation Rapide

L'extension est en mode développement. Voici comment l'installer en moins de 60 secondes :

1.  **Téléchargez** le projet (Bouton vert `Code` > `Download ZIP`).
