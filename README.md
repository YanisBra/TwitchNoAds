# TwitchFreeTurbo 🚀

TwitchFreeTurbo est une extension Chrome moderne conçue pour améliorer votre expérience sur Twitch en éliminant les interruptions publicitaires et en ajoutant des fonctionnalités bonus indispensables.

## ✨ Fonctionnalités

- **Anti Pre-rolls** : Bloque les publicités dès le chargement d'un stream.
- **Anti Mid-rolls** : Contourne les publicités pendant le direct via la technologie "Thunderdome".
- **Mode Incrustation (SDA)** : Si une publicité forcée apparaît, le stream est automatiquement remis en grand et la pub réduite dans un coin.
- **Dernière Chance** : Un bouton ⏪ ajouté sur le lecteur Twitch pour reculer instantanément de 10 secondes.
- **Auto-Points** : Récupère automatiquement vos points de chaîne (bonus de coffre).
- **Statistiques** : Suivez en temps réel le nombre de publicités bloquées.
- **Design Twitch** : Une interface moderne parfaitement intégrée au style sombre de Twitch.

## 🛠️ Installation

Comme l'extension n'est pas encore sur le Chrome Web Store, voici comment l'installer manuellement :

1.  **Téléchargez** ce dépôt (cliquez sur "Code" > "Download ZIP") et extrayez-le sur votre ordinateur.
2.  Ouvrez **Google Chrome**.
3.  Allez dans la gestion des extensions en tapant `chrome://extensions/` dans la barre d'adresse.
4.  Activez le **"Mode développeur"** en haut à droite.
5.  Cliquez sur le bouton **"Charger l'extension dépaquetée"** (Load unpacked).
6.  Sélectionnez le dossier contenant les fichiers de l'extension.
7.  **C'est prêt !** N'oubliez pas d'épingler l'extension dans votre barre d'outils.

## 🔧 Utilisation

- Cliquez sur l'icône de l'extension pour ouvrir le menu de réglages.
- Par défaut, tout est activé pour une protection immédiate.
- Utilisez le bouton "Tester la notification" pour vérifier le bon fonctionnement visuel.

---
*Note : Cette extension est fournie à titre éducatif. Twitch améliore constamment ses systèmes, l'extension est mise à jour régulièrement pour rester fonctionnelle.*
